# Ape-Escape-Tracker

Tracker for Ape Escape, for use with AP

Tracker version 0.3.2

Installation instructions:

- Download [Poptracker](https://github.com/black-sliver/PopTracker/releases)
- Place **Ape-Escape-Tracker.zip** in packs folder
- Run Poptracker
